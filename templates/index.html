Documentación de API

Endpoint para crear un usuario

            
Endpoint: POST {url_servidor}/usuarios

Ejemplo de solicitud:
POST /usuarios HTTP/1.1
Host: api.example.com
Content-Type: application/json

{
    "cedula_identidad": "123456789",
    "nombre": "Juan",
    "primer_apellido": "Pérez",
    "segundo_apellido": "Gómez",
    "fecha_nacimiento": "1990-01-01"
}

Respuesta exitosa:
HTTP/1.1 201 Created
Content-Type: application/json

{
    "message": "Usuario creado",
    "id_usuario": 1
}

Endpoint: GET {url_servidor}/usuarios

Ejemplo de solicitud:
GET /usuarios HTTP/1.1

        "segundo_apellido": "Gómez",
        "fecha_nacimiento": "1990-01-01"
    },
    ...
]

Endpoint para obtener un usuario en específico

    
Endpoint: GET {url_servidor}/usuarios/{id_usuario}

Ejemplo de solicitud:
GET /usuarios/1 HTTP/1.1
Host: api.example.com

Respuesta exitosa:
HTTP/1.1 200 OK
Content-Type: application/json

{
    "id_usuario": 1,
    "cedula_identidad": "123456789",
    "nombre": "Juan",
    "primer_apellido": "Pérez",
    "segundo_apellido": "Gómez",
    "fecha_nacimiento": "1990-01-01"
}

Endpoint para actualizar los datos de un usuario

    
Endpoint: PUT {url_servidor}/usuarios/{id_usuario}

Ejemplo de solicitud:
PUT /usuarios/1 HTTP/1.1
Host: api.example.com
Content-Type: application/json

{
    "cedula_identidad": "987654321",
    "nombre": "Pedro",
    "primer_apellido": "González",
    "segundo_apellido": "López",
    "fecha_nacimiento": "1985-05-10"
}

Respuesta exitosa:
HTTP/1.1 200 OK
Content-Type: application/json

{
    "message": "Usuario actualizado"
}

Endpoint para eliminar a un usuario

    
Endpoint: DELETE {url_servidor}/usuarios/{id_usuario}

Ejemplo de solicitud:
DELETE /usuarios/1 HTTP/1.1
Host: api.example.com

Respuesta exitosa:
HTTP/1.1 200 OK
Content-Type: application/json

{
    "message": "Usuario eliminado"
}
